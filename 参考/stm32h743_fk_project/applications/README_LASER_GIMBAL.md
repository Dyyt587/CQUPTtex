# MS4010双电机激光云台控制系统

## 概述

基于MS4010双电机实现的二维激光云台控制系统，支持通过输入距离和XY坐标来精确控制激光指向任意位置。

## 文件结构

```
applications/
├── laser_gimbal.h              # 激光云台头文件
├── laser_gimbal.c              # 激光云台实现
├── laser_gimbal_test.c         # 测试程序和命令行接口
├── laser_gimbal_example.c      # 使用示例和演示
├── LASER_GIMBAL_USAGE_GUIDE.md # 详细使用指南
└── drv_ms4010.h/c             # MS4010电机驱动 (依赖)
```

## 核心功能

✅ **坐标控制**: 输入距离和XY坐标，自动计算角度并控制激光指向  
✅ **三维坐标支持**: 完整的三维空间坐标控制  
✅ **角度控制**: 直接设置偏航角和俯仰角  
✅ **平滑运动**: 可配置的速度和加速度参数  
✅ **安全限位**: 可设置的角度限制范围  
✅ **多种模式**: 直接控制、坐标控制、跟踪等模式  
✅ **线程安全**: 完整的多线程安全设计  
✅ **命令行接口**: 便于调试和测试的MSH命令

## 快速开始

### 1. 硬件连接
- 偏航轴电机 (ID=1): 水平旋转控制
- 俯仰轴电机 (ID=2): 垂直旋转控制  
- RS485通信: 默认uart2, 115200波特率

### 2. 系统初始化
```shell
# 命令行初始化
msh> laser_init

# 或运行完整演示
msh> laser_gimbal_quick_demo
```

### 3. 基本使用
```shell
# 坐标控制 - 指向距离10米，前方8米，右侧3米的位置
msh> laser_point 10.0 8.0 3.0

# 角度控制 - 偏航30度，俯仰15度
msh> laser_angle 30.0 15.0

# 回零
msh> laser_home

# 查看状态
msh> laser_status
```

### 4. 程序调用示例
```c
#include "laser_gimbal.h"

// 初始化后，指向目标点
laser_gimbal_point_to_coordinate(&gimbal, 10.0f, 8.0f, 3.0f);

// 或直接设置角度
laser_gimbal_set_angle(&gimbal, 30.0f, 15.0f);
```

## 坐标系说明

```
        Y轴 (向右为正)
        ↑
        |
Z轴 ----+---→ X轴 (向前为正)
(向上为正)

偏航角 (Yaw): 绕Z轴旋转，向右为正 (-180° ~ +180°)
俯仰角 (Pitch): 绕Y轴旋转，向下为正 (-30° ~ +80°)
```

## 命令参考

| 命令 | 参数 | 功能 |
|------|------|------|
| `laser_init` | - | 初始化激光云台系统 |
| `laser_point` | `<距离> <x> <y>` | 坐标控制指向 |
| `laser_angle` | `<偏航> <俯仰>` | 角度控制 |
| `laser_home` | - | 回零操作 |
| `laser_status` | - | 查看状态 |
| `laser_test` | `basic\|scan\|precision` | 运行测试 |
| `laser_stop` | - | 停止运动 |

## 应用示例

### 固定点监控
```shell
# 监控大门 (距离20米，前方18米，右侧5米)
laser_point 20.0 18.0 5.0

# 监控停车场 (距离15米，前方10米，左侧8米)  
laser_point 15.0 10.0 -8.0
```

### 区域扫描
```shell
# 运行水平扫描测试
laser_test scan

# 或自定义扫描
laser_angle -45.0 10.0
laser_angle -30.0 10.0
laser_angle -15.0 10.0
# ... 继续扫描
```

### 安防巡逻
```shell
# 运行安防巡逻演示
security_patrol_example
```

## 性能参数

- **控制精度**: ±0.01° (MS4010电机精度)
- **最大速度**: 可配置 (默认30-45°/s)
- **响应时间**: < 100ms
- **控制距离**: 0.5m ~ 50m
- **角度范围**: 偏航±120°, 俯仰-20°~+60°

## 故障排除

1. **初始化失败**: 检查RS485连接和电机ID (需要ID=1,2)
2. **角度超限**: 调整目标坐标或角度限制参数
3. **通信异常**: 使用`ms4010_test status`检查电机状态
4. **精度问题**: 调整机械参数和零点偏移

## 技术特点

- **自动角度解算**: 根据坐标自动计算最优角度
- **平滑运动控制**: 可配置的速度和加速度曲线
- **多重安全保护**: 角度限位、参数验证、异常处理
- **丰富的接口**: 支持命令行、程序调用、回调等多种方式
- **实时状态反馈**: 完整的状态监控和统计信息
