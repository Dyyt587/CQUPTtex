# 激光云台陀螺仪补偿功能使用指南

## 概述
激光云台陀螺仪补偿功能可以根据陀螺仪检测到的载体运动来实时补偿云台的姿态，保持激光指向的稳定性。这对于安装在移动平台上的云台系统特别有用。

## 功能特点

### 1. 实时补偿
- 50Hz高频补偿更新
- 自动检测载体运动并进行反向补偿
- 支持yaw和pitch轴独立补偿控制

### 2. 零偏校准
- 自动陀螺仪零偏校准功能
- 支持手动设置零偏值
- 校准数据持久化保存

### 3. 可调节补偿增益
- yaw轴和pitch轴独立增益调节
- 增益范围：0.0-2.0
- 推荐初始值：0.3-0.5

### 4. 智能补偿策略
- 只在载体运动较大时才应用补偿
- 避免与速度控制模式冲突
- 角度限位保护

## 使用方法

### 第一步：陀螺仪零偏校准
在使用陀螺仪补偿功能之前，必须先进行零偏校准：

```bash
# 1. 确保云台完全静止
# 2. 开始校准（采集60个样本）
laser_gyro_calibrate 60

# 3. 等待采样完成后，完成校准
laser_gyro_finish_cal

# 4. 查看校准结果
laser_gyro_status
```

**重要提示：**
- 校准过程中必须保持云台完全静止
- 避免振动和干扰
- 校准环境应该稳定

### 第二步：设置补偿增益
根据实际需要调整补偿增益：

```bash
# 设置适中的补偿增益
laser_gyro_gain 0.3 0.3

# 或者更强的补偿（适用于运动剧烈的平台）
laser_gyro_gain 0.5 0.5
```

**增益调节指南：**
- 0.1-0.3：轻微补偿，适用于微小振动
- 0.3-0.5：中等补偿，适用于一般移动平台
- 0.5-1.0：强补偿，适用于剧烈运动平台
- 1.0以上：谨慎使用，可能导致过补偿

### 第三步：启用补偿功能
```bash
# 启用陀螺仪补偿
laser_gyro_enable 1

# 查看运行状态
laser_gyro_status
```

### 第四步：测试和调优
1. 让载体进行小幅度运动
2. 观察云台补偿效果
3. 根据需要调整增益参数
4. 重复测试直到达到满意效果

## 故障排除

### 常见问题

**Q: 补偿效果不明显**
A: 
1. 检查增益设置是否太小
2. 确认陀螺仪数据是否正常
3. 验证校准是否正确

**Q: 出现过补偿现象**
A:
1. 降低补偿增益
2. 重新进行零偏校准
3. 检查载体振动是否过大

**Q: 校准失败**
A:
1. 确保校准期间云台完全静止
2. 增加校准采样数量
3. 检查陀螺仪连接和数据

**Q: 补偿方向错误**
A:
1. 检查坐标系映射是否正确
2. 调整增益符号
3. 验证陀螺仪安装方向

### 状态指示
- **ENABLED**: 补偿功能已启用
- **DISABLED**: 补偿功能已禁用
- **Calibrating**: 正在进行零偏校准
- **Data Invalid**: 陀螺仪数据无效

## 最佳实践

### 部署建议
1. **初始设置**: 从较小的增益开始（0.2-0.3）
2. **逐步调优**: 根据实际效果逐步增加增益
3. **环境测试**: 在不同运动条件下测试补偿效果
4. **定期校准**: 建议定期重新校准零偏

### 使用场景
- ✅ **移动车辆**: 补偿车辆行驶时的颠簸
- ✅ **船舶平台**: 补偿海浪引起的摇摆
- ✅ **无人机载荷**: 补偿飞行器的姿态变化
- ✅ **手持设备**: 减少手持抖动影响

### 限制条件
- ❌ **高频振动**: 超过50Hz的振动无法有效补偿
- ❌ **大角度运动**: 超过云台机械限位的运动
- ❌ **高速旋转**: 超过±60°/s的快速运动
